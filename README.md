# top_admin-dashboard

An admin dashboard project from [The Odin Project](https://www.theodinproject.com/) curriculum built with HTML and CSS.

## Features

- Sidebar navigation
- User profile header
- Project cards with action buttons
- Announcements section
- Trending users section

## Technologies Used

- HTML5
- CSS3 (Grid & Flexbox)
